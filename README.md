### LinkedIn Auto Apply
An automation scaffold for guided LinkedIn job search using Selenium and Chrome remote-debugging session reuse.

**Why this exists:**
- browser automation is more reliable than fragile screen-based clicking
- dedicated session reuse is cleaner than hijacking a normal profile
- the goal is controlled automation with clear stop conditions, not chaos

**Current direction:**
- Easy Apply focused
- Session reuse over brittle UI hacks
- Safe exits for weird/custom/external flows

**Built with:** Python, Selenium, Chrome remote debugging
